## Технический план

### Часть 1: Табы навигации (5 мин)

**Файл**: `styles.css` (строки 789-794)

**Текущий код** (`.tab-btn.active`):
```css
.tab-btn.active {
    color: var(--accent-primary);
    background: white;  /* ← УБРАТЬ */
    border: 1px solid rgba(200, 184, 160, 0.4);  /* ← ИЗМЕНИТЬ */
    box-sizing: border-box;
}
```

**Новый код**:
```css
.tab-btn.active {
    color: var(--accent-primary);
    background: transparent;  /* ✅ Прозрачный фон */
    border: 1.5px solid #C8B8A0;  /* ✅ Яркий beige без прозрачности */
    box-sizing: border-box;
}
```

**Изменения**:
- `background: white;` → `background: transparent;`
- `border: 1px solid rgba(200, 184, 160, 0.4);` → `border: 1.5px solid #C8B8A0;`

---

### Часть 2: Заголовок с угловой рамкой (10 мин)

**Файл**: `styles.css` (строки 91-94)

**Текущий код** (`.screen-title`):
```css
.screen-title {
    font-size: 32px;
    font-weight: 700;
    color: var(--text-primary);
    margin: 0 0 24px 0;
    line-height: 1.2;
}
```

**Новый код**:
```css
.screen-title {
    position: relative;  /* ✅ Для псевдоэлементов */
    font-size: 32px;
    font-weight: 700;
    color: var(--text-primary);
    margin: 0 0 24px 0;
    line-height: 1.2;
    padding: 16px 0;  /* ✅ Отступы для рамки */
}

/* [T-009] Угловая рамка - верхний левый угол */
.screen-title::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    width: 50px;
    height: 50px;
    border-top: 2.5px solid #C8B8A0;
    border-left: 2.5px solid #C8B8A0;
}

/* [T-009] Угловая рамка - нижний правый угол */
.screen-title::after {
    content: '';
    position: absolute;
    bottom: 0;
    right: 0;
    width: 50px;
    height: 50px;
    border-bottom: 2.5px solid #C8B8A0;
    border-right: 2.5px solid #C8B8A0;
}
```

**Изменения**:
- Добавить `position: relative;`
- Добавить `padding: 16px 0;`
- Создать `::before` (верхний левый угол)
- Создать `::after` (нижний правый угол)

---

## Edge Cases

### 1. Проверка конфликтов псевдоэлементов
**Действие**: Проверить что `.screen-title` не использует `::before`/`::after`
```bash
grep -n "\.screen-title::" styles.css
```
**Ожидание**: Пусто (нет конфликтов)

### 2. Media query для малых экранов
**Действие**: Проверить media query для `.screen-title`
```css
@media (max-width: 380px) {
    .screen-title {
        font-size: 28px;
    }
}
```
**Проверка**: Угловая рамка должна корректно работать при уменьшенном шрифте
