#### Функция: `onServiceSelect(serviceId)`

**Назначение**: Обработчик выбора услуги — загружает доступные даты, теперь также автоматически устанавливает нужный месяц.

**Добавляемая логика** (после `hideLoader()`, до `renderBookingScreen()`):

```javascript
// [T-011] Автопереключение: если в текущем месяце нет слотов — идём на первый доступный
if (State.availableDates.length > 0) {
    const y = State.currentMonth.getFullYear();
    const m = State.currentMonth.getMonth();
    const hasCurrentMonthSlots = State.availableDates.some(d => {
        const [, dm, dy] = d.date.split('.');
        return parseInt(dy) === y && parseInt(dm) - 1 === m;
    });
    if (!hasCurrentMonthSlots) {
        const [, fm, fy] = State.availableDates[0].date.split('.');
        State.currentMonth = new Date(parseInt(fy), parseInt(fm) - 1, 1);
    }
}
```

**Обработка ошибок**:
- `availableDates` пустой → блок не выполняется
- Слоты есть в текущем месяце → `currentMonth` не меняется
- Формат даты отличается от `DD.MM.YYYY` → parseInt вернёт NaN, Date создаст Invalid Date; в этом случае отображение некорректное, но не крашится

## Потоки данных

### Основной поток (Happy Path) — Автопереключение

1. Пользователь выбирает услугу
   ↓
2. `onServiceSelect(serviceId)`: очищает State, `currentMonth = new Date()` (текущий месяц)
   ↓
3. `loadAvailableDates(serviceId)`: загружает даты, заполняет `State.availableDates`
   ↓
4. Блок автопереключения: проверяет есть ли слоты в `currentMonth`
   - Слотов нет → `State.currentMonth = new Date(fy, fm-1, 1)` (первый доступный месяц)
   - Слоты есть → `currentMonth` не изменяется
   ↓
5. `renderBookingScreen()`: рендерит календарь с корректным месяцем
   ↓
6. Пользователь видит даты сразу

## Решения и компромиссы

### Архитектурные решения

**Решение 1: Где размещать логику автопереключения**
- **Проблема**: Где добавить проверку — в `loadAvailableDates` или в `onServiceSelect`?
- **Рассмотренные варианты**:
  - Вариант А: внутри `loadAvailableDates` — меньше code scatter, но смешивает ответственности (загрузка данных + UI state)
